# Portfolio Website

This is a simple personal portfolio website built with plain HTML, CSS, and JavaScript.

## Run it locally

Open `/Users/saishriyasurla/Desktop/Projects/portfolio/index.html` in your browser.

## Customize it

Edit these files:

- `index.html` for your name, about text, project details, and links
- `style.css` for colors, spacing, and design
- `script.js` for the small scroll animation

## Free hosting options

### GitHub Pages

1. Create a GitHub repository named `portfolio`
2. Upload these files
3. In GitHub, open `Settings > Pages`
4. Under `Build and deployment`, choose `Deploy from a branch`
5. Select the `main` branch and `/ (root)`
6. Save, then wait a minute or two

Your site will be live for free on a GitHub Pages URL.

### Netlify

1. Push this folder to GitHub
2. Log in to Netlify with GitHub
3. Import the repo
4. Deploy with default settings

Netlify also has a generous free plan for static sites.
